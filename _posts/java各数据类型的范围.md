``` java
Boolean ：true、false
Char 	2个字节Byte (16bit) 
byte    1个字节Byte (8bit)     -2^7~2^7-1      -128 ~ 127
short   2个字节Byte (16bit)    -2^16~2^16-1    -32768 ~ 32767			
int     4个字节Byte (32bit)    -2^31~2^31-1    -2147483648 ~ 2147483647    Integer.MIN_VALUE ~ Integer.MAX_VALUE
long    8个字节Byte (64bit)    -2^63~2^63-1    -9223372036854775808 ~ 9223372036854775807
float   4个字节Byte (32bit)    1.17549435E-38 ~ 1.4E-45
double  8个字节Byte (64bit)    4.9E-324 ~ 1.7976931348623157E308
```
